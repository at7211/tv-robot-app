三步驟開始使用
簡單到連豬都會用
1

安裝並啟動

下載 .pkg 安裝檔，安裝後從 Launchpad 啟動。Menu bar 會出現一個遊戲控制器的圖示。

2

掃描 QR Code

點擊 menu bar 圖示，彈出 QR Code。用手機掃描後，自動開啟遙控器網頁。電腦與手機需在同一個 Wi-Fi 網路下。

3

開始操控！

手機瞬間變成你的遙控器。播放、暫停、調音量、切全螢幕，甚至可以遠端操控滑鼠，全部一指搞定。

常見問題
需要開通哪些系統權限？
需要在「系統設定 → 隱私權與安全性 → 輔助使用」中，允許「肥豬電腦遙控器」控制你的電腦。這是 macOS 的安全機制，app 需要此權限才能模擬鍵盤和滑鼠操作。

路徑：系統設定 → 隱私權與安全性 → 輔助使用，找到肥豬電腦遙控器並打勾。
安裝時出現「無法打開，因為無法驗證開發者」？
因為 app 沒有 Apple 開發者簽名。請到「系統設定 → 隱私權與安全性」，在下方找到被阻擋的提示，點擊「強制打開」即可。或者在 Finder 裡對 app 按右鍵選擇「打開」。
手機跟電腦需要在同一個網路嗎？
是的，手機和電腦必須連接到同一個 Wi-Fi 網路。App 會在區域網路內啟動一個 HTTP 伺服器，手機透過這個伺服器來傳送控制指令。
支援哪些 macOS 版本？
macOS 10.15 (Catalina) 及以上版本皆可使用。
手機需要安裝 app 嗎？
不需要！手機只要用任何瀏覽器掃描 QR Code，就會打開遙控器網頁。iPhone、Android 都可以用，完全不用安裝任何東西。
可以控制哪些影音平台？
理論上所有在電腦上播放的影音都可以控制，包括 Netflix、YouTube、Disney+ 等。因為 app 是模擬鍵盤和滑鼠操作，不限定特定平台。
